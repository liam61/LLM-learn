## Agent 是什么

AI Agents 是基于 LLM，能够自主理解、自主规划决策、执行复杂任务的智能体。Agents 不是 chatGPT 的升级版，它不仅告诉你如何做，更会帮你去做

AI Agents = LLM + 规划技能 + 记忆 + 工具使用

![alt text](../../images/agentzc.png)

本质上，Agents 是一个 LLM 的编排和执行系统

## Agents 如何实现

![alt text](../../images/agentrhsx.png)

1. 提出需求/问题
2. 问题 + prompt 组合
3. ReAct Loop
4. 查找 Memory
5. 查找可用工具
6. 执行工具并观察结果
7. 得到之中结果

可以将 2~6 黑盒化
